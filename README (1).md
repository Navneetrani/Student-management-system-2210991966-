## Project Description

A Python-based console application that automates student attendance tracking using CSV file storage. The system supports adding students, marking daily attendance (Present/Absent), viewing attendance reports, and calculating individual attendance percentages — with no internet connection or additional hardware required.

---

## Technologies Used

- Python 3.10
- csv Module (built-in)
- datetime Module (built-in)
- os Module (built-in)
- VS Code (Development)
- Git / GitHub (Version Control)

---

## Key Features

1. Add students with roll number, name, and branch
2. Mark daily attendance (Present / Absent) with auto-captured date
3. View complete attendance report in tabular format
4. Calculate individual attendance percentage per student
5. Data persistence via CSV files across sessions
